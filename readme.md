# 说明

这是一个基于`fastify`的`nodejs`服务器，用于将脚本发送过来的数据发送到`notion`的数据库中。

# 配置
在`.env`中配置`notion`的`token`和`database_id`。

# 使用
```shell
npm install
node server.js
```
